### üöÄ Getting Started

#### Prerequisites

- Node.js 16+ and npm/yarn/pnpm
- Modern browser with support for CSS backdrop-filter

#### Installation

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Run tests
npm test
```
